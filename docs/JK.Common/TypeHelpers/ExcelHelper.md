# ExcelHelper

**Namespace:** `JK.Common.TypeHelpers`

Provides helper methods for working with Excel-related data.

### GetColumnName

**Signature:** ``GetColumnName(Int32@ columnNumber)``

**Summary:**
Converts a column number to its corresponding Excel column name (e.g., 1 = "A", 27 = "AA").

**Parameters:**
- **columnNumber** — The column number to convert.

**Returns:** The corresponding Excel column name.
