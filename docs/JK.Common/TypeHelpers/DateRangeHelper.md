# DateRangeHelper

**Namespace:** `JK.Common.TypeHelpers`

Helper methods for working with date ranges.

### DoesOverlap

**Signature:** ``DoesOverlap(DateTime@ startOne, DateTime@ endOne, DateTime@ startTwo, DateTime@ endTwo)``

**Summary:**
Determines whether or not two date ranges overlap.

**Parameters:**
- **startOne** — The start date of range one.
- **endOne** — The end date of range one.
- **startTwo** — The start date of range two.
- **endTwo** — The end date of range two.

**Returns:** True if overlap, otherwise false.
