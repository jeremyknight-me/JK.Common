# AlphanumericSpecification

**Namespace:** `JK.Common.Specifications`

Specification to determine if a string is alphanumeric.

## IsSatisfiedBy

**Signature:** `IsSatisfiedBy(String candidate)`

**Summary:**
Determines whether the specified candidate string is alphanumeric.

**Parameters:**

- **candidate** — The string to evaluate as alphanumeric.

**Returns:** **true** if the candidate is alphanumeric; otherwise, **false** .
