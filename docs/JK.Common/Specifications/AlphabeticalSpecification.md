# AlphabeticalSpecification

**Namespace:** `JK.Common.Specifications`

Specification to determine if a string is alphabetical.

## IsSatisfiedBy

**Signature:** `IsSatisfiedBy(String candidate)`

**Summary:**
Determines whether the specified candidate string is alphabetical.

**Parameters:**

- **candidate** — The string to evaluate as alphabetical.

**Returns:** **true** if the candidate is alphabetical; otherwise, **false**.
