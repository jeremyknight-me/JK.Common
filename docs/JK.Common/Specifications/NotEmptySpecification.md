# NotEmptySpecification

**Namespace:** `JK.Common.Specifications`

Specification to determine if a string is not null, empty, or whitespace.

## IsSatisfiedBy

**Signature:** `IsSatisfiedBy(String candidate)`

**Summary:**
Determines whether the specified candidate string is not null, empty, or whitespace.

**Parameters:**

- **candidate** — The string to evaluate.

**Returns:** **true** if the candidate is not null, not empty, and not whitespace; otherwise, **false** .
