# InternetProtocolAddressSpecification

**Namespace:** `JK.Common.Specifications`

Specification to determine if a string is a valid IPv4 address.

## IsSatisfiedBy

**Signature:** `IsSatisfiedBy(String candidate)`

**Summary:**
Determines whether the specified candidate string is a valid IPv4 address.

**Parameters:**

- **candidate** — The string to evaluate as an IPv4 address.

**Returns:** **true** if the candidate is a valid IPv4 address; otherwise, **false** .
