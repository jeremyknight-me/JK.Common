# PhoneNumberSpecification

**Namespace:** `JK.Common.Specifications.UnitedStates`

Specification to determine if a string is a valid United States phone number.

### IsSatisfiedBy

**Signature:** ``IsSatisfiedBy(String@ candidate)``

**Summary:**
Determines whether the specified candidate string is a valid United States phone number.

**Parameters:**
- **candidate** — The string to evaluate as a phone number.

**Returns:** **true** if the candidate is a valid United States phone number; otherwise, **false** .
