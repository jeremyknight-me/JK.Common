# EmailSpecification

**Namespace:** `JK.Common.Specifications`

Specification to determine if a string is a valid email address.

## IsSatisfiedBy

**Signature:** `IsSatisfiedBy(String candidate)`

**Summary:**
Determines whether the specified candidate string is a valid email address.

**Parameters:**

- **candidate** — The string to evaluate as an email address.

**Returns:** **true** if the candidate is a valid email address; otherwise, **false** .
